CSS Snippets for Tables

  • When adding a CSS property to a table using the table ID:
    • If the property is for the whole table use table#tableID 
      • table#table3{
}
    • If the property is for the <tr> or <th> or <td> use table#tableID th 
      • table#table3 th{
}
    • If the property is for odd/even columns or rows use table#tableID:nth child(odd) 
      • table#tableID:nth child(odd){
}
    • If the property is for a specific row or column use table#tableID td:nth child(2).
      This will change the 2nd of the <td> 
      • table#table3 td:nth child(2){
}

Merging Cells

Example
HTML
CSS
Fruit Quantity
Apple 5
Cherries
Banana 10
Orange 7
4 
<table id="table7">
    <tr>
        <th>Fruit</th>
        <th>Quantity</th>
    </tr>
    <tr>
        <td>Apple</td>
        <td>5</td>
    </tr>
    <tr>
        <td class="merge_col" colspan="2"
            >Cherries</td>
    </tr>
    <tr>
        <td>Banana</td>
        <td>10</td>
    </tr>
    <tr>
        <td class="merge_row" rowspan="2"
            >Orange</td>
        <td>7</td>
    </tr>
    <tr>
        
        <td>4</td>
    </tr>
</table>
table#table7,
table#table7 td {
    border: 2px solid black;
    padding: 15px;
}
.merge_col,
.merge_row {
    text-align: center;
}

Table Header Default CSS

th {
    display: table-cell;
    vertical-align: inherit;
    font-weight: bold;
    text-align: center;
}

Table Row Default CSS

tr {
    display: table-row;
    vertical-align: inherit;
    border-color: inherit;
}

Table Cell Default CSS

td {
    display: table-cell;
    vertical-align: inherit;
}
