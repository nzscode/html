Description Lists

  • <dl> defines a description List
  • <dt> defines the term Name
  • <dd> defines the description
Example
HTML
Agar
A Jelly like dish made using seaweed
Custard
A soft pudding made using cornstarch
<dl>
    <dt>Agar</dt>
    <dd>A Jelly like dish made using seaweed</dd>
    <dt>Custard</dt>
    <dd>A soft pudding made using cornstarch</dd>
</dl>

Removing Bullets

Removing <li> bullets: 
ul {
    list-style-type: none;
}

Reducing and Expanding line heights

Example
HTML
CSS
  1. Bread
  2. Cereal
  3. Egg


  1. Bread
  2. Cereal
  3. Egg
<ol class="line_height50">
    <li>Bread</li>
    <li>Cereal</li>
    <li>Egg</li>
</ol>
<br /><br />
<ol class="line_height300">
    <li>Bread</li>
    <li>Cereal</li>
    <li>Egg</li>
</ol>
.line_height50 {
    all: revert;
    line-height: 50%;
}

.line_height300 {
    all: revert;
    line-height: 300%;
}

Changing Bullet Color

  • Bullet color can only be changed if bullet is removed, then a new bullet can be added and colored.
Example
HTML
CSS
  • Harry
  • Ron
  • Hermione
<ul class="bullet_color">
    <li>Harry</li> 
    <li>Ron</li>
    <li>Hermione</li>
</ul>
ul.bullet_color {
    list-style: none;
}
ul.bullet_color li::before {
    content: "* ";
    color: blue;
}

Default CSS for Ordered Lists 

ol {
    display: block;
    list-style-type: decimal;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 0;
    margin-right: 0;
    padding-left: 40px;
}

Default CSS for Unordered Lists 

ul {
    display: block;
    list-style-type: disc;
    margin-top: 1em;
    margin-bottom: 1 em;
    margin-left: 0;
    margin-right: 0;
    padding-left: 40px;
}
