Description Lists

  • <dl> defines a description List
  • <dt> defines the term Name
  • <dd> defines the description
Example
HTML

                                
Agar

                                
A Jelly like dish made using seaweed

                                
Custard

                                
A soft pudding made using cornstarch

                            
<dl>
    <dt>Agar</dt>
    <dd>A Jelly like dish made using seaweed</dd>
    <dt>Custard</dt>
    <dd>A soft pudding made using cornstarch</dd>
</dl>

Default CSS for Ordered Lists

ol {
    display: block;
    list-style-type: decimal;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 0;
    margin-right: 0;
    padding-left: 40px;
}

Default CSS for Unordered Lists

ul {
    display: block;
    list-style-type: disc;
    margin-top: 1em;
    margin-bottom: 1 em;
    margin-left: 0;
    margin-right: 0;
    padding-left: 40px;
}

Default CSS for Description Lists

dl {
    display: block;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 0;
    margin-right: 0;
}

Default CSS for List Items

li {
    display: list-item;
}

Default CSS for Description Term

dt {
    display: block;
    margin-left: 40px;
}

Default CSS for Description Definition 

dd {
    display: block; 
}
