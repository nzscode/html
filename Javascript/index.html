Arrow Function

    1. Must have parentheses
    2. Function keyword is not used
    3. Inside the parentheses will be the parameters
    4. After the parentheses, comes an arrow
    5. Then comes a code block, followed by a semi-colon.
    6. for one parameter, the parentheses can be removed.
Code without Arrow Function 
const add = function(a, 3){
    return a + b;
}
Code using Arrow Function 
const add = (a, 3) => {
    return a + b;
};
Code with one parameter 
const add = a => {
    return a + 3;
}
Code with one return statement 
const add = a => a + 3;
