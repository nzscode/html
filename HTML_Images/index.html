Basic Image

  • The tag is empty.
  • Contains attributes only, and does not have a closing tag.
  • Has two required attributes:
    • src - Specifies the path to the image
    • alt - Specifies an alternate text for the image
  • Can also have a size attribute:
    • Width & Height can be set on the image using the style attribute (Recommended)
    • Width & Height can also be set through the Stylesheet. If setting via Stylesheet, remeber stylesheets are not hardcoded, whereas the style attribute is. To use Stylesheet, follow the CSS given
Example
HTML
CSS

<img 
    src="../images/img.png" 
    alt="Star Image"
    style="width: 25px; 
            height: 25px" />
img {
    width: 25px; 
    height: 25px
}
