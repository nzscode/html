  • Tuples - (Round Brackets)
    • Ordered
    • Unchangeable
    • Indexed
    • Can have duplicates
    • Can store String("anything within single/double quotations"), int(Integers/Floats), and Boolean(True/False) data types 
    myTuple = (2.358, "Robby", False, "John", "Lisa", "Li -Mae", "Carey", ("Onions, Potatoes"))
    • Constructing a Tuple
      • If creating a tuple with only one item, a comma must be used after, if not it will NOT be considered a tuple.
      • Creating a tuple by variable declaration
        • myTuple = ()
print(type(myTuple))
        # Output:
<class 'tuple'>
      • Using a tuple() Constructor
        • newTuple = tuple("a", 1, True)
    • Accessing Tuple Items
      • Using the index number
        • print(myTuple[1])
        # Output: 
Robby
      • Using a range of Index Numbers
        • print(f"Length of Tuple : {len(myTuple)}")
print(f"Items from index 3 to the end {myTuple[3:]}")
print(f"Items from the start to index 4{myTuple[:4]}")
print(f"Items from index 2 to the 5{myTuple[2:5]}")
        # Output:
Length of Tuple : 8
Items from index 3 to the end ('John', 'Lisa', 'Li -Mae', 'Carey', 'Onions, Potatoes')
Items from the start to index 4(2.358, 'Robby', False, 'John')
Items from index 2 to the 5(False, 'John', 'Lisa')
    • Checking if an item exists
      • if "Carey" in myTuple:
print("Carey is in myTuple")
      # Output: 
Carey is in myTuple
    • Change the values of the tuple
      1. Since a tuple is unchangeable, first convert the tuple to a list 
        2. tupleList = list(myTuple)
      2. Now change the item at the required index
        3. tupleList[3] = "Anna"
      3. Now convert the list back into a tuple(Can replace original tuple, with this new tuple) 
        4. myTuple = tuple(tupleList)
      tupleList = list(myTuple)
tupleList[3] = "Anna"
myTuple = tuple(tupleList)
print(myTuple)
      # Output: 
(2.358, 'Robby', False, 'Anna', 'Lisa', 'Li -Mae', 'Carey', 'Onions, Potatoes')
    • Adding a tuple to a tuple
      • newTuple = (5, "Amy Adams", ["ten", "eleven"])
myTuple += newTuple
print(myTuple)
      # Output:
(2.358, 'Robby', False, 'John', 'Lisa', 'Li -Mae', 'Carey', 'Onions, Potatoes', 5, 'Amy Adams', ['ten', 'eleven'])
    • Removing items from a Tuple
      1. Since a tuple is unchangeable, first convert the tuple to a list 
        2. tupleList = list(myTuple)
      2. Now remove the item.
        • When removing an item that is made up of multiple items, you have to list them all.(See below) 
        tupleList.remove("Onions, Potatoes")
      3. Now convert the list back into a tuple(Can replace original tuple, with this new tuple) 
        4. myTuple = tuple(tupleList)
      tupleList = list(myTuple)
tupleList.remove("Onions, Potatoes")
myTuple = tuple(tupleList)
print(myTuple)
      # Output: 
(2.358, 'Robby', False, 'John', 'Lisa', 'Li -Mae', 'Carey')
    • Deleting a Tuple
      • del myTuple
print(myTuple)
      • Raises an error since the tuple no longer exists. 
      # Output: 
NameError: name 'myTuple' is not defined. Did you mean: 'tuple'?
    • Unpacking a Tuple
      • In Python, values from within the tuple can be extracted out and assigned to variables.
      • The number of variables must match the number of Tuples.
        • If you want more than one item to belong to the variable, you must place a * in front of the variable.
        • If there is more than one set of data within the tuple, that has more than one element assigned to it,
          then the data must be unpacked together as one variable, and then break it down with lists.
          1. Assign the tuple data to variables:
            2. cost, teacher, bring_lunch, *students_and_assistants, bus_company = tuple_data
          2. Break down the combined tuple data into their own variables, using indexes 
            3. students = students_and_assistants[:4]
assistant_teachers = students_and_assistants[4:6]
          tuple_data = (50, "Mr. Smith", True, "John", "Alice", "Bob", "Emily", "Ms. Johnson", "Mr. Brown", "ABC Bus Company")

cost, teacher, bring_lunch, *students_and_assistants, bus_company = tuple_data

students = students_and_assistants[:4]
assistant_teachers = students_and_assistants[4:6]

print("Cost:", cost)
print("Teacher:", teacher)
print("Bring lunch:", bring_lunch)
print("Students:", students)
print("Assistant Teachers:", assistant_teachers)
print("Bus Company:", bus_company)
      myTuple = (2.358, "Robby", False, "John", "Lisa", "Li -Mae", "Carey", ("Onions, Potatoes"))
(cost, teacher, bring_lunch, *students, groceries) = myTuple
print(f"Cost = {cost}\n"
f"Teacher = {teacher}\n"
f"Bring Lunch = {bring_lunch}\n"
f"Students = {students}\n"
f"Groceries = {groceries}")

    • Tuple Methods
      • count ()
        • Counts how many times a given item occurs in the tuple 
        myTuple = ("John", "Lisa", "Li -Mae", "Carey", "John", "Yusuf", "John", "Tracy")
print(myTuple.count("John"))
        # Output: 
3
      • index()
        • Searches for the first index 
        myTuple = ("John", "Lisa", "Li -Mae", "Carey", "John", "Yusuf", "John", "Tracy")
print(myTuple.index("John"))
        # Output: 
0
