table

CSS Snippets for Tables

Adding CSS using table ID

For the Whole Table

  • If the property is for the whole table use table#tableID
CSS
table#table3{
                }


For the <tr>|<th>|<td>

  • If the property is for the <tr> or <th> or <td> use table#tableID th
Default CSS
table#table3 th{
                }


For the odd|even columns|rows

  • If the property is for odd/even columns or rows use table#tableID:nth child(odd)
Default CSS
table#tableID:nth child(odd){
                }


For a specific row|column

  • If the property is for a specific row or column use table#tableID td:nth child(2).
  • This will change the 2nd of the <td>
Default CSS
table#table3 td:nth child(2){
                }

Merging Cells

Examples
HTML
Fruit Quantity
Apple 5
Cherries
Banana 10
Orange 7
4 
<table id="table7">
    <tr>
        <th>Fruit</th>
        <th>Quantity</th>
    </tr>
    <tr>
        <td>Apple</td>
        <td>5</td>
    </tr>
    <tr>
        <td class="merge-col" colspan="2"
            >Cherries</td>
    </tr>
    <tr>
        <td>Banana</td>
        <td>10</td>
    </tr>
    <tr>
        <td class="merge-row" rowspan="2"
            >Orange</td>
        <td>7</td>
    </tr>
    <tr>
        
        <td>4</td>
    </tr>
</table>
CSS
table#table7,
table#table7 td {
    border: 2px solid black;
    padding: 15px;
}
.merge-col,
.merge-row {
    text-align: center;
}


Creating a table with a caption

  • Used to specify whether the borders of table cells should be collapsed into a single border or not.
  • When border-collapse is set to collapse , adjacent cell borders will be merged into a single border, resulting in a cleaner and more compact table design
  • This is the default value
