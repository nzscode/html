Ordered Lists

  • Defines an ordered list
  • Can be numerical or alphabetical
  • <li> tag is used to define each list item
Example
HTML
Default CSS
  1. Mercury
  2. Venus
  3. Earth
  4. Mars
  5. Jupiter
  6. Saturn
  7. Uranus
  8. Neptune
<ol>
    <li>Mercury</li>
    <li>Venus</li>
    <li>Earth</li>
    <li>Mars</li>
    <li>Jupiter</li>
    <li>Saturn</li>
    <li>Uranus</li>
    <li>Neptune</li>
</ol>
ol {
    display: block;
    list-style-type: decimal;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 0;
    margin-right: 0;
    padding-left: 40px;
}

li {
    display: list-item;
}

Reversed

  • A boolean attribute
  • Specifies that the list order should be descending (9,8,7...), instead of ascending (1, 2, 3...)
Example
HTML
  1. Cat
  2. Dog
  3. Antelope
  4. Baboon
<ol reversed>
    <li>Cat</li>
    <li>Dog</li>
    <li>Antelope</li>
    <li>Baboon</li>
</ol>

Start

  • Specifies the start value of the first list item in an ordered list
  • Always an integer, even when the numbering type is letters or romans. E.g., to start counting list items from the letter "c" or the roman number "iii", use start="3"
Example
HTML
  1. Cat
  2. Dog
  3. Antelope
  4. Baboon
<ol start="25">
    <li>Cat</li>
    <li>Dog</li>
    <li>Antelope</li>
    <li>Baboon</li>
</ol>

Type

  • Specifies the kind of marker to use in the list (letters or numbers)
  • CSS list-style-type property offers more types
Example
HTML
  1. Cat
  2. Dog
  3. Antelope
  4. Baboon


  1. Cat
  2. Dog
  3. Antelope
  4. Baboon
<ol type="I">
    <li>Cat</li>
    <li>Dog</li>
    <li>Antelope</li>
    <li>Baboon</li>
</ol>

<ol type="a">
    <li>Cat</li>
    <li>Dog</li>
    <li>Antelope</li>
    <li>Baboon</li>
</ol>

Unordered Lists

  • Defines an unordered (bulleted) list
  • <li> tag is used to define each list item
  • Can use CSS list-style-type property to change bullet type
Example
HTML
Default CSS
  • Onion
  • Garlic
  • Potatoes
  • Green Peppers


  • Onion
  • Garlic
  • Potatoes
  • Green Peppers


  • Onion
  • Garlic
  • Potatoes
  • Green Peppers
<ul>
    <li>Onion</li>
    <li>Garlic</li>
    <li>Potatoes</li>
    <li>Green Peppers</li>
</ul>
ul {
    display: block;
    list-style-type: disc;
    margin-top: 1em;
    margin-bottom: 1 em;
    margin-left: 0;
    margin-right: 0;
    padding-left: 40px;
}

li {
    display: list-item;
}

Square bullets CSS 

.ul-list-style-type-square {
    list-style-type: square;
}

Custom Bullet CSS 

ul.ul-list-style-type-custom {
    list-style: none;
    margin-left: -30px;
}

ul.ul-list-style-type-custom li::before {
    content: "+*+";
    color: hotpink;
}

Description Lists

  • <dl> defines a description List
  • <dt> defines the term Name
  • <dd> defines the description
Example
HTML
Default CSS
Agar
A Jelly like dish made using seaweed
Custard
A soft pudding made using cornstarch
<dl>
    <dt>Agar</dt>
    <dd>A Jelly like dish 
            made using seaweed</dd>
    <dt>Custard</dt>
    <dd>A soft pudding 
            made using cornstarch</dd>
</dl>
dl {
    display: block;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 0;
    margin-right: 0;
}

dt {
    display: block;
    margin-left: 40px;
}

dd {
    display: block; 
}

Reducing and Expanding line heights

Example
HTML
CSS
  1. Bread
  2. Cereal
  3. Egg


  1. Bread
  2. Cereal
  3. Egg
<ol class="line-height50">
    <li>Bread</li>
    <li>Cereal</li>
    <li>Egg</li>
</ol>
<br /><br />
<ol class="line-height300">
    <li>Bread</li>
    <li>Cereal</li>
    <li>Egg</li>
</ol>
.line-height50 {
    all: revert;
    line-height: 50%;
}

.line-height300 {
    all: revert;
    line-height: 300%;
}

Changing Bullet Color

  • Bullet color can only be changed if bullet is removed, then a new bullet can be added and colored.
Example
HTML
CSS
  • Harry
  • Ron
  • Hermione
<ul class="bullet-color">
    <li>Harry</li> 
    <li>Ron</li>
    <li>Hermione</li>
</ul>
ul.bullet-color {
    list-style: none;
}
ul.bullet-color li::before {
    content: "* ";
    color: blue;
}
